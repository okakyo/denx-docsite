# DENX アーカイブへようこそ

## DENX アーカイブとは

DENX　アーカイブとは、これまでのDENX の勉強会で使用したパワーポイントや資料、知識として共有するために作成した。

## 特徴
- Markdown を使った記事の管理
- Vite (Vue 3) を駆使したブログの設計

## 使用した技術
- **Vitepress** : フレームワーク
- ESLint,prettier : Linter
- Vercel : デプロイサーバー
